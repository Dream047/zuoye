# 多目标物品自动计数与位置标注系统

> 基于MATLAB图像处理工具箱的视觉计数与定位解决方案

本项目实现了一套完整的图像目标自动检测系统，能够对散落物品（如硬币、瓶盖、电子元器件等）图像进行预处理、分割、计数和个体定位，并输出可视化标注结果与坐标数据。

---

## 📌 项目功能

- **自动计数**：统计图像中所有独立目标的总数量
- **目标定位**：计算每个目标的质心坐标（像素级精度）
- **可视化标注**：在原图上绘制外接矩形框及编号标签
- **结果导出**：将坐标信息保存为文本文件，便于后续分析
- **批量处理**：支持对文件夹内多张图像进行自动批处理

## 📊 实验结论
核心性能指标：
| 指标 |测试结果
| 计数准确率 |	≥ 95%（目标分离良好时）
| 单张处理耗时 |	< 0.5 秒（1024×768 分辨率）
| 定位精度	| 亚像素级（质心坐标）
| 适用目标类型	|硬币、瓶盖、圆形/规则零件

方法优势：
✅ 无需训练：基于传统图像处理，免去数据标注与模型训练成本
✅ 高效轻量：计算复杂度低，适合嵌入式或实时场景
✅ 结果可解释：每一步处理具有明确的物理/几何意义

已知局限：
⚠️ 对目标严重粘连或重叠的情况处理效果不佳（会合并为一个区域）
⚠️ 对光照不均匀或阴影较重的图像分割质量下降
⚠️ 形态学结构元素（strel）大小需根据目标尺寸手动适配

针对上述局限，可通过引入分水岭算法（解决粘连）或自适应局部阈值（解决光照）进行改进。
